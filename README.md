### Installation:

```
$ git clone https://github.com/mahynski/FHMCSimulation.git
$ cd FHMCSimulation/build
$ cmake CMakeLists.txt; make
```

---

### Unittests

```
$ cd FHMCSimulation/tests/src
$ cmake CMakeLists.txt; make
$ cd ../bin; ./runUnitTests
```

---

### Dependencies

1. GCC
2. C++11 support required
