\subsection{Detailed Description}
Abstract base class which defines all pair potentials. 

Note that tail corrections should be \char`\"{}double\char`\"{} the \char`\"{}standard tail corrections\char`\"{} which try to account for double-\/counting ahead of time. The system only iterates over unique pairs, so do N\-O\-T do this. Also, when doing M\-C insertions/deletions this double counting is never a problem so easier to do it this way.
