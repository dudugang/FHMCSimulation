double lennardJones::tailCorrection ( const double  rhoBath)
virtual

Calculate the tail correction with the approximation g(r) = 1 for r_{cut} > 1 as explained in Frenkel & Smit in eq.

(3.2.5). Tail corrections only account for number of fully inserted particles so I have chosen not to scale this part of the energy with expanded ensemble stage.
